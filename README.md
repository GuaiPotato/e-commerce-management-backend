# e-commerce-management-backend

> 本项目为本人vue课程期末课程大作业。
>
> 该项目中没有进行持久数据存储，只是为了熟悉vue的使用。
>
> 如果有兴趣，可自行添加后台管理，接入数据库等操作。

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve

### Compiles and minifies for production
```

```

### Lints and fixes files
```
